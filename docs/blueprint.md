# **App Name**: UCN Connecta

## Core Features:

- Secure User Authentication & Registration: Implement a robust registration system for graduates and companies, including administrative verification and approval workflows. This features secure login capabilities and strong password enforcement for all user types.
- Dynamic Graduate Profile Management: Graduates can create, view, and update their professional profiles, detailing academic background, work experience, achievements, and success stories, fostering a professional network environment.
- Company Profile & Job Posting: Approved companies can establish institutional profiles and post job vacancies, making them accessible to the graduate community. This includes dedicated forms for company and administrator registration.
- Alumni Network & Discovery: Features include publicly accessible graduate profiles, institutional statistics display (e.g., graduates by career, year), and an interactive geographical map illustrating graduate distribution by department.
- Admin Portal for Content & User Moderation: A comprehensive administrative dashboard allowing staff to verify graduate and company registrations, manage user accounts (approval, rejection, deletion with audit trail), moderate content, and export data reports (Excel, CSV, PDF) with filtering options.
- AI-Powered Description Assistant: Integrate a tool using generative AI to assist graduates and companies in drafting compelling and professional descriptions for their profiles or job postings, providing suggested content based on input keywords.

## Style Guidelines:

- Primary color: A deep, institutional blue (#1F3C7E), evoking trust and professionalism, serving as a cornerstone for brand identity and interactive elements.
- Background color: A very light, almost off-white hue (#E9EBF0), offering a clean and airy canvas that ensures readability and a sense of calm and clarity for a professional context.
- Accent color: A vibrant cyan-blue (#21B7E0), providing a modern, dynamic contrast for call-to-action buttons, highlights, and interactive feedback, enhancing user engagement without being overpowering.
- Headlines will utilize 'Space Grotesk' (sans-serif), lending a modern, tech-savvy yet professional feel suitable for drawing attention. Body text will use 'Inter' (sans-serif), chosen for its excellent readability across various screen sizes and its neutral, objective aesthetic, pairing well with the headline font.
- Use a consistent set of clean, modern line-art or outline icons throughout the application. These should be minimalistic to maintain a professional and uncluttered appearance, complementing the overall UI design.
- Adopt a fully responsive and adaptive design approach, ensuring optimal viewing and interaction across all devices. Employ a modular, card-based layout for displaying graduate profiles, job listings, and news, facilitating easy scanning and information digestion. Prioritize clear visual hierarchy and ample whitespace.
- Incorporate subtle, tasteful animations for transitions between sections, hover effects on interactive elements, and loading indicators. Animations should be swift and functional, enhancing the user experience without causing distractions or delays.
